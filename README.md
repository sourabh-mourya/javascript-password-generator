
 
# 🔐 Password Generator

A simple yet powerful **Password Generator** that allows users to create **secure and customizable passwords** in just one click.  
Perfect for safeguarding your accounts with strong, random passwords!

---

## ✨ Features

✅ **Adjustable Password Length** — Set length between **6 to 20 characters**.  
✅ **Customizable Character Set** — Choose from:
- Uppercase Letters (A–Z)  
- Lowercase Letters (a–z)  
- Numbers (0–9)  
- Special Symbols (!, @, #, $, %, etc.)  

✅ **Real-time Slider** — Adjust password length dynamically.  
✅ **Instant Generation** — Create a new password with a single click.  
✅ **Clean & Responsive UI** — Works on all screen sizes.


---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **No frameworks** — Pure, lightweight, and fast
